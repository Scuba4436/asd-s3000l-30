# ASD S3000L Issue 3.0

Dieses Repository enthält das Datenpaket und die offizielle Spezifikation für **ASD S3000L Issue 3.0** (International procedure for Logistics Support Analysis - LSA).

## Inhalt

- **`S3000L_Issue_3.0.pdf`**: Das offizielle S3000L Issue 3.0 Spezifikationsdokument.
- **`s3000l_3-0_data_model_001-00.eap`**: Sparx Enterprise Architect (EA) Projektdatei, die das UML-Datenmodell enthält.
- **`s3000l_3-0_data_model_001-00.xmi`**: XML Metadata Interchange (XMI) Export des UML-Datenmodells, geeignet für den Import in andere Modellierungswerkzeuge.
- **`XML Schema Package/`**: Enthält die XML Schema Definition (XSD) Dateien für den S3000L Datenaustausch.
  - `s3000l_3-0_lsaDataset.xsd`: Hauptschema für LSA-Datensätze.
  - `s3000l_3-0_valid_values*.xsd`: Schemata zur Definition von gültigen Werten (Valid Values), Einheiten und Codelisten.
  - `s3000l_3-0_project_specific_*.xsd`: Schemata für projektspezifische Austauschdefinitionen und Erweiterungen.

## Über S3000L

Die ASD S3000L-Spezifikation definiert die Prozesse und Informationsanforderungen für die Durchführung der Logistics Support Analysis (LSA) während des Lebenszyklus eines Systems. Sie bietet einen strukturierten Ansatz zur Identifizierung und Dokumentation der logistischen Unterstützungsressourcen, die für den Betrieb und die Wartung eines Produkts in seiner vorgesehenen Umgebung erforderlich sind.

Weitere Informationen finden Sie auf der offiziellen [S-Series Website](https://www.sx000i.org/) oder der [S3000L Website](https://www.s3000l.org/).
